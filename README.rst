Installation
------------

You can install ``pyxu-eigh`` via `pip`_:

.. code-block:: bash

   pip install pyxu-eigh


To install latest development version :

.. code-block:: bash

   pip install git+https://github.com/AIR-Hub/pyxu-eigh.git
